# 🐕 Robo Dog – Low-Cost Quadruped Robot

## 📌 Overview

Robo Dog is a compact, low-cost quadrupedal robot designed to demonstrate basic concepts of autonomous robotics using simple and accessible components. The project integrates sensing, processing, and actuation to create a functional robotic system capable of movement and interaction.

The robot uses an **Arduino Nano** as the main controller, **MG995 servo motors** for leg movement, and an **HC-SR04 ultrasonic sensor** for object detection.

---

## 🎯 Objectives

* Design and build a quadruped robot with 12 degrees of freedom
* Implement basic locomotion (forward and backward movement)
* Integrate sensor-based interaction using an ultrasonic sensor
* Develop a cost-effective and educational robotics platform

---

## ⚙️ Hardware Components

* Arduino Nano (ATmega328P)
* MG995 Servo Motors (12x)
* HC-SR04 Ultrasonic Sensor
* Power Supply (Battery Pack)
* Frame (Sunboard / Prototype Body)

---

## 💻 Software

* Arduino IDE
* Programming Language: C/C++

---

## 🚀 Features

* Basic quadruped locomotion
* Ultrasonic-based object detection
* Tail wagging behavior when object is detected
* Modular and customizable design

---

## ⚠️ Limitations

* Mechanical instability due to flexible frame
* Limited to basic movement (no advanced gait)
* No full autonomous navigation implemented
* Limited processing power of Arduino Nano

---

## 📊 Results

* Successfully built a working quadruped robot
* Achieved basic forward and backward motion
* Implemented sensor-based reactive behavior (tail wagging)

---

## 🌱 Future Improvements
